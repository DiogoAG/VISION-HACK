# **App Name**: ShowUp

## Core Features:

- Guest Event Display: Guests can view event details such as name, date, time, venue, vibe description, and a cover photo placeholder. This view also displays the current RSVP count.
- Guest RSVP Flow: Guests enter their first name to RSVP. Upon successful submission, a confirmation message 'You're on the list 🎉' is displayed. This information is stored in the host's local browser storage.
- Guest RSVP Availability Status: Based on the host's RSVP toggle, guests will see either the RSVP button or a 'Sorry, this event is full' message.
- Host Dashboard: Event Details Editor: Hosts can view and edit event details (title, date, venue, vibe) directly from their dashboard. Changes are saved to local storage.
- Host Dashboard: Live Guest List: The dashboard displays a live list of RSVP'd guests, each as a card with their name and the time they RSVP'd, all stored in local browser storage.
- Host Dashboard: Headcount & RSVP Toggle: A running headcount of RSVPs is displayed, along with a toggle switch to open or close RSVPs for the event, affecting what guests see in real time on their device.
- Vibe Description Suggester: A generative AI tool in the host dashboard to suggest creative 'vibe descriptions' for the event, helping the host articulate the party's atmosphere.

## Style Guidelines:

- The visual concept of 'Bold and hype' for a party app dictates a dark, vibrant, and energetic color scheme. The background will be a near-black with a subtle violet tint (#1A171D).
- The primary color, 'electric violet' (#9947EB), will be used for key interactive elements and prominent text. It's chosen for its bold, energetic saturation and good contrast against the dark background.
- The accent color, 'hot pink' (#EF0F8F), is analogous to the primary and offers a striking contrast for call-to-action buttons, highlights, and confirmations, embodying the 'hype' aesthetic.
- The 'Space Grotesk' sans-serif font is recommended for both headlines and body text. Its contemporary, tech-inspired aesthetic aligns with both the 'real event app' feel of the guest view and the 'clean command center' of the host dashboard.
- Playful and modern flat or duotone icons that complement the bold color palette should be used to enhance the 'hype' and engaging feel, especially in the guest view.
- The design will be mobile-first, ensuring large, easily tappable elements for guests. The guest page will feature visually engaging, dynamic layouts, while the host dashboard will maintain a clean, organized, and data-centric command center structure.
- Subtle, celebratory animations will be incorporated for guest RSVP confirmations and state changes (e.g., toggle activations in the host dashboard), adding to the 'hype' and enhancing user feedback.
